# 扣哥公众号写作指南

## 账号定位

**人设**：过来人/老炮——踩过坑、见过真金白银，说话直接不留情面，但句句在理。

**核心价值**：认知觉醒+焦虑唤醒。不是教炒股，是教人用投资思维做人生决策，提前感知风口，不再当"赌徒"。

**目标人群**：所有想赚钱的人。

**品牌口号**：每天一个认知，关注我，别再错过。

## 行文风格

### 核心原则
- **制造焦虑和紧迫感**：让读者意识到自己一直在"投资"（工作、婚姻、买房、生娃、选行业），但从来不计算ROI
- **戳破自我安慰**：直接撕开"我没亏只是没赚""等等看再说"的心理防线
- **老炮口吻**：不讨好读者，不写正确的废话，每句话都要有刺痛感
- **用数据说话**：具体数字、具体倍数、具体场景，不写空泛的道理

### 写作禁忌
- 不写"建议""可以""或许"这类软绵绵的词
- 不写正确的废话（如"投资有风险""要理性决策"）
- 不做老好人式的平衡论述
- 不用"首先、其次、最后"这种学术式结构

### 语言特征
- 短句为主，节奏感强
- **少用问句和反问句**——问句多了像在审问读者，说教味很重。陈述事实比质问更有力量
- **少用"你"密集出现**——"你"可以用，但不能每段都以"你"开头，不能连续出现。读者需要感觉是在听一个故事，不是在被点名教育
- 善用对比制造荒诞感（如买房被夸稳重 vs 炒股被骂赌博）
- 关键结论用**加粗+红色**标出
- 以观察者和讲述者的姿态写作，让事实和逻辑自己说话

### 三大技法融合（半佛仙人×刘润×余哥牛弹琴）

扣哥的风格 = 半佛的拆解刀法 + 刘润的内容框架 + 余哥的情绪浓度，三者融合而非照搬。

#### 技法一：利益链条拆解法（学半佛仙人）

**核心**：任何现象背后都有一条利益链条，写文章就是把这条链条一环一环拆给读者看。

**拆解四步法**：
1. **定义利益方**：这件事里谁赚钱？谁买单？谁被割？
2. **追溯资金流**：钱从谁的口袋流向谁的口袋？中间经过几道手？
3. **识别信息差**：利益方之间的信息不对称在哪？谁故意制造信息差？
4. **暴露荒诞性**：把利益链条摊开后，读者会发现自己一直在为别人的利润买单

**扣哥适配**：半佛拆商业骗局，扣哥拆人生决策的隐性成本。买房、选行业、跳槽、创业——每个决策背后都有一条利益链条，大多数人看不到。

**句式示例**：
- "你以为你在选行业？不，你在为别人的退出通道买单。"
- "这笔钱经过了三道手才到你面前，每道手抽走20%，你算过没有？"
- "表面上是教育投资，拆开看是恐惧变现。"

#### 技法二：深度+娱乐双满足（学半佛仙人）

**核心**：读者不是不想看长文，是不想看无聊的长文。深度和可读性不矛盾。

**实现方法**：
- **类比降维**：用读者熟悉的场景解释复杂概念（"选行业就像选赛道，你在一条堵死的路上踩油门有什么用？"）
- **黑色幽默**：在严肃分析中插入荒诞类比，制造"笑完冒冷汗"的效果
- **段子化表达**：观点不变，但表达要有梗。每500字至少一个让人会心一笑或倒吸冷气的表达
- **"先爽后痛"节奏**：先用有趣的切入让读者读下去，然后在中段用数据拍醒

#### 技法三：SCQA内容框架（学刘润）

**核心**：每篇文章都是一个"问题→解决"的完整闭环，用SCQA确保逻辑清晰。

**SCQA四步**：
- **S（Situation/情境）**：读者身处的真实场景。"你工作5年，存了30万，想搞点投资。"
- **C（Complication/冲突）**：打破这个场景的变量。"但你发现，30万放5年，购买力缩水了15%。"
- **Q（Question/问题）**：冲突引出的核心问题。"普通人到底该怎么跑赢通胀？"
- **A（Answer/答案）**：扣哥的核心观点和方法论。

**扣哥适配**：SCQA是骨架，利益链拆解是血肉，情绪浓度是皮肤。先用SCQA理清逻辑，再用拆解法填充内容，最后用情绪表达包装。

**SCQA不是死板模板**：不需要在文中标出SCQA，而是用它检查文章逻辑是否完整。每篇文章写完后自查：情境有没有？冲突够不够痛？问题够不够尖锐？答案够不够实？

#### 技法四：情绪浓度拉满（学余哥牛弹琴）

**核心**：文章不是论文，是战斗檄文。每个段落都要有情绪温度。

**情绪浓度四级**：
- **Level 1 陈述**："房价涨了30%。"（纯信息，无情绪）
- **Level 2 带刺**："房价涨了30%，你的工资涨了吗？"（反问制造落差）
- **Level 3 戳心**："房价涨了30%，你的工资没涨，你还觉得自己没亏？"（直接攻击自我安慰）
- **Level 4 暴击**："房价涨了30%，你的工资原地踏步，你每天加班12小时的结果是——帮房东多赚了一套车位。"（荒诞化+具象化）

**扣哥标准**：正文平均情绪浓度不低于Level 3，核心段落必须达到Level 4。开篇和结尾必须是Level 4。

**合规护栏**：情绪浓度拉满不等于造谣攻击。高情绪必须建立在真实数据和逻辑之上。暴击感来自"事实本身足够荒诞"，而不是"夸大事实制造荒诞"。

**禁区**：
- 不对具体个人进行人身攻击（攻击现象和行为，不攻击人）
- 不用民族主义情绪煽动（余哥这方面有时越线，扣哥不学这个）
- 不编造数据制造恐慌（所有数据必须可查）

#### 技法五：一鱼多吃内容复用（学刘润）

**核心**：一个核心观点至少产出3种内容形态，最大化内容ROI。

**复用路径**：
- **深度长文**（公众号主文，1200-2000字）→ 完整论述
- **热点速评**（日更短文，500-800字）→ 核心观点+当日热点结合
- **金句卡片**（朋友圈/社群传播）→ 文中最戳心的一句话做成图片
- **评论区互动话题**→ 从文章争议点提取讨论问题

**实操**：写完一篇深度文章后，立即提取3条可独立传播的金句、1个可讨论的争议观点、1个可对接明日热点的切入角度。

## 文章结构模板

### 标题
- **主标题**：狠、短、有冲击力，最好是一个判断句或反问句
- **副标题**：补充说明文章主题，带悬念感
- 示例：《你才是那个赌徒》《你的钱是怎么没的》《你不醒，钱就是别人的》

### 正文结构（不超过5-6个章节，SCQA骨架+拆解血肉+情绪皮肤）

**底层逻辑**：SCQA确保逻辑完整，利益链拆解确保内容有料，情绪浓度确保读者读完。

1. **开篇钩子（S+C）**：用两个震撼数字或一个荒诞场景建立情境（S），然后立刻引入冲突（C）。情绪浓度≥Level 4。
   - 示例："2025年，AI替代了43万个白领岗位（S）。但最先失业的不是能力最差的，而是工资最高的那批（C）。"
2. **重新定义（Q）**：打破读者固有认知，提出真正的问题。这是SCQA中的Q——一个读者从没想过但一听就睡不着的问题。
   - 用利益链拆解法：这件事里谁赚钱？谁买单？为什么你一直看不到这条链条？
3. **撕破伪装（拆解链条）**：把利益链条一环一环拆开，对比读者的荒诞行为。情绪浓度≥Level 3。
   - 每个环节给数据、给对比、给"笑完冒冷汗"的类比
4. **算账环节（数据暴击）**：用具体数据算几笔让人睡不着觉的账。这是利益链拆解法的高潮——把所有隐性成本显性化。
   - 每笔账必须精确到具体金额/百分比/倍数
5. **核心观点（A）**：SCQA的Answer——扣哥的判断和方法论。升华到投资思维的高度。情绪浓度≥Level 4。
   - 用一句可独立传播的金句封装核心观点（一鱼多吃的素材来源）
6. **CTA引流**：制造损失感+关注引导。情绪浓度≥Level 4。

**自检清单**（写完每篇文章后过一遍）：
- [ ] S有没有？读者能不能在3秒内代入情境？
- [ ] C够不够痛？冲突是否直接戳到读者的处境？
- [ ] Q够不够尖锐？问题是否让读者"从没想过但一听就慌"？
- [ ] A够不够实？答案是否能提炼成一句可传播的金句？
- [ ] 利益链条拆了没有？读者看完是否知道"钱去了哪"？
- [ ] 情绪浓度够了没？开篇和结尾是否达到Level 4？
- [ ] 有没有至少1个"笑完冒冷汗"的类比或段子？

## 排版规范

- **必须输出DOCX格式**，不用HTML或Markdown

- 输出文件保存到 `outputs/` 目录

**视觉层级**：
- 主标题：40号字，加粗，居中
- 一级标题：32号字，加粗
- 二级标题：28号字，加粗

- `dataCard()`：数据卡片，灰色底，红色高亮关键数据

- `makeTable()`：数据表格，表头灰色底

**颜色规范**：
- 红色 `B5493A`：负面数据、警示、关键结论
- 绿色 `52C41A`：正面数据、机会
- 灰色 `666666`：辅助信息
- 粉色底 `FFF5F5`：警示类内容
- 蓝色底 `F0F7FF`：对比类内容

## 微信公众号合规红线

**必须遵守平台规则，文章可以狠，但不能越线。**

### 六大违规类型（绝对不碰）
1. **政治敏感**：不涉及政治人物、政治事件、政府政策评价
2. **色情低俗**：不写招嫖、情色、性暗示内容
3. **虚假谣言**：所有数据和事件必须有据可查，不编造、不夸大
4. **商业欺诈**：不推荐股票、不喊单、不传销、不网赚
5. **恶意诱导**：不用"不转不是中国人"等胁迫分享，不用集赞抽奖利诱
6. **侵权冒用**：不冒用他人身份，不盗用他人内容

### 标题党红线
- 标题可以狠，但不能**虚假夸大**或**与内容严重不符**
- 标题可以是判断句/反问句，但不能捏造事实
- 例：《你才是那个赌徒》✅（内容确实在讲这个）；《震惊！某某死了》❌（标题党）

### 处罚机制（阶梯式）
- 初犯：删文警告或停用部分功能3天
- 较重：封号7天
- 90天内累计3次：封号15-30天
- 传播谣言、欺诈、多次严重违规：**永久封号，无申诉机会**

### 内容写作底线
- 只讲事实，不编造、不歪曲
- 可以评论，但不人身攻击
- 可以尖锐，但不造谣传谣
- 涉及具体人物时用"某"字脱敏（如"余某""杨某"）
- 涉及具体学校/企业时可脱敏（如"武汉某高校""某手机品牌"），由用户决定是否脱敏

### 金融类内容特别注意
- 严禁违规放贷、催收、虚拟货币交易推广
- 不侮辱、威胁任何人
- 投资类内容必须加免责声明："以上内容仅作信息整理与分析，不构成任何投资建议"

## 脱敏处理

文章涉及具体企业名称、人物名字、学校名称时，**行文前询问用户是否需要脱敏**。

脱敏级别：
- **轻度脱敏**：人名用"某"（余某、杨某），企业保留全称
- **中度脱敏**：学校用"某高校"，企业用"某品牌"
- **重度脱敏**：所有具体名称都用代称

用户说保留的例外保留（如SK海力士）。

## 热点评论类文章写作指南

当文章涉及社会热点事件、争议话题时：

### 写作原则
1. **只讲事实**：事件经过、时间线、各方回应，客观陈述
2. **不站队**：可以分析影响，但不做道德审判
3. **不人身攻击**：评论行为，不攻击个人
4. **有据可查**：所有引用的数据、言论必须有来源

### 评论类文章结构
1. **开篇**：节日祝福/事件切入，快速进入主题
2. **还原事件**：时间线+各方回应，只陈述事实
3. **引出深层问题**：从单个事件延伸到普遍现象
4. **核心观点**：分析影响和启示，不攻击具体人
5. **升华**：回归到认知层面，给读者启发
6. **CTA引流**：关注引导

### 可用话题方向
- 品牌营销翻车事件分析
- 流量经济的扭曲现象
- 互联网记忆与口碑管理
- 高校/企业的公关危机
- 社会热点事件的深层逻辑

## 热点结合

文章要蹭实时热点和数据：
- AI相关新闻（英伟达市值、AI芯片动态、大模型发布）
- 股市/基金数据
- 社会热点事件
- 风口轮动数据（移动互联网→短视频→直播→AI）
- 薪资/通胀/房价数据

使用 `hot-topics` skill 获取当前热搜，或搜索最新数据确保准确性。

## 微信公众号推荐策略（流量获取核心）

### 一、推荐算法机制（2025-2026）

**双引擎模型**：从"订阅优先"转向"算法推荐+社交裂变"双引擎，非粉丝流量占比显著提升。

**三维推荐模型**：
- 社交关系：好友点赞("朋友♡")权重最高，真实社交裂变可让新号突破粉丝限制
- 兴趣标签：NLP标签匹配，账号内容必须垂直，避免标签混乱
- 行为数据：打开率(40%) + 互动率(30%) + 分享率(20%) + 完读率(10%)

**四大推荐入口**：
- "看一看"热点页：侧重时效性与标题冲击力
- 底部"喜欢此内容的人还喜欢"：依赖内容相关性和标签匹配
- 合集标签页：聚集垂直流量
- 订阅号"朋友♡"栏：靠社交互动触发

**流量池赛马机制**：新文章进入冷启动曝光池（500-1000曝光），平台根据数据表现决定是否推入下一级流量池。在时间窗口期内数据靠前的文章会获得第二次推流。

### 二、搜索SEO策略（搜一搜排名）

**关键词布局**：
- 公众号名称和简介中包含核心关键词
- 文章标题中包含1次核心关键词
- 正文中自然出现3-5次关键词
- 关键词密度控制在1%-3%之间
- 优先选取长尾词降低竞争

**标题SEO原则**：
- 标题必须包含用户会搜索的词（如"A股""牛市""4200点"）
- 不要只用花哨的文学标题，要覆盖搜索意图
- 数字+事件+判断句结构最优（如"A股突破4200创11年新高，这次是真牛市"）

**前200字权重**：
- 微信搜索对文章前200字赋予更高权重
- 核心关键词必须在前200字内出现
- 开篇就进入主题，不要铺垫废话

**账号权重因素**：
- 认证账号排名优于未认证账号
- 持续运营时长影响权重
- 原创占比（80%以上原创最优）
- 更新频率（每周2-3篇保持活跃度）

### 三、标题优化策略

**高CTR标题公式**：
- 数字+对比：如"A股突破4200创11年新高"
- 身份+内幕：如"操盘手不会告诉你的3个信号"
- 痛点+承诺：如"你亏钱的真正原因，不是运气"
- 反常识+悬念：如"牛市里亏最多的，是这类人"
- 利益链暴露（学半佛）：如"你的房贷利息，养活了哪些人？"
- 荒诞对比（学半佛）：如"你花3年选老公，花3分钟选行业"
- SCQA浓缩（学刘润）：如"存了5年的30万，购买力缩水了多少？"
- 直接判断（学余哥）：如"这个行业，3年内必死"
- 拆穿话术（学半佛）：如"他们说'稳赚不赔'的时候，你的钱已经在路上了"

**标题三重检验**：
1. **搜索检验**：标题中有没有用户会搜的关键词？
2. **情绪检验**：读者看到标题是否会产生焦虑/好奇/不服？
3. **信息检验**：标题是否传递了一个明确的判断或事实？

**标题禁忌**：
- 不要纯文学化（"大A这回真支棱起来了"无搜索量）
- 不要与内容严重不符（平台惩罚）
- 不要用"震惊""速看"等低质标题党词汇

**标题长度**：控制在15-25字，太短没信息量，太长被截断

### 四、完读率提升策略

**开篇3段决定生死**：
- 痛点共鸣开场（"你有没有遇到过..."）
- 震撼数据开场（"4200点，11年新高"）
- 悬念钩子开场（直接抛结论让人想看论证）

**文章长度控制**：
- 日更热点类：800-1500字最优
- 深度分析类：1200-2000字
- 新手起号期：控制在500-1000字

**排版"呼吸感"**：
- 短段落，每段2-3句话
- 不要用大段文字墙
- 关键数据加粗或标红
- 段间留白，视觉不拥挤

**节奏控制**：
- 每200-300字设一个小高潮（数据、金句、反问）
- 避免长篇铺垫，直接给料
- "信息密度要高，废话密度要零"

### 五、互动率提升策略

**评论引导（文末钩子）**：
- 抛出开放式问题："你觉得A股这波能到4500吗？评论区说说"
- 制造争议："我说这是牛市，你信不信？"
- 投票式引导："看好的扣1，看空的扣2"

**中间互动钩子**：
- 文中埋入半句话，让读者想评论补充
- 说出有争议的观点，激发讨论欲
- "信息差"写法：说一半留一半

**提升"在看"和分享**：
- 打造"社交货币"：提供谈资或独家数据
- 文末引导："点击在看让同事也看到"
- 金句设计：让读者想转发给朋友的那句话

**评论区运营**：
- 冷启动期每条评论都回复
- 把精彩评论置顶增加互动感
- 禁止刷赞刷评（平台会过滤并降权）

### 六、发布时间策略

**黄金时间窗口**：
- 早高峰：7:00-9:00（通勤碎片时间）
- 午间：12:00-13:30（午休刷手机）
- 晚高峰：18:00-20:00（下班放松期）
- 深夜：22:00-23:30（睡前阅读）

**热点类文章**：越早发越好，抢时间窗口。热点过了再发等于废文。

**时间窗口与赛马**：发布后1-2小时内的数据表现决定是否进入下一级流量池。因此发布后要立刻做冷启动动作（分享到私域群、朋友圈）。

### 七、冷启动与新号起量

**新手保护期**：新号前10-20篇有平台流量扶持，务必保持：
- 内容垂直，只写一个方向
- 每篇都原创
- 分享到朋友圈和相关群
- 回复每条评论

**冷启动动作清单**：
- 发布后立刻转发朋友圈（第一波基础阅读量）
- 分享到2-3个高活跃度社群
- 引导群内真实互动（不要求赞，而是抛话题引发讨论）
- 不做任何刷量行为（平台过滤+降权）

**垂直定位要求**：
- 前10篇内容必须在同一垂直领域
- 让平台准确为你打标签
- 标签一旦混乱，推荐精准度大幅下降

### 八、内容策略总则

**日更热点类文章检查清单**：
- [ ] 标题包含可搜索关键词
- [ ] 前200字出现核心关键词
- [ ] 文章字数800-1500字
- [ ] 开篇3段有钩子（数据/悬念/痛点）
- [ ] 文中有1-2个互动钩子
- [ ] 文末有评论引导问题
- [ ] 内容垂直，不跑题
- [ ] 发布时间在黄金窗口内
- [ ] 热点时效性OK（当天热点当天发）
- [ ] 摘要包含关键词和核心信息

**摘要（文章简介）写法**：
- 包含核心关键词（提升搜索曝光）
- 概括文章最吸引人的信息点
- 50-120字，简洁有料
- 示例："沪指突破4200创11年新高，成交额连续破3万亿。2026牛市主线方向：算力、存储芯片、商业航天。"

**系列化内容更易爆款**：
- 同一话题拆成系列文章
- 每篇末尾预告下期内容
- 系列内部形成内链
- 增加用户粘性和关注动力

### 九、数据复盘与迭代

**每周必看数据**：
- 推荐流量占比（后台→数据→内容分析→流量分析）
- 各篇完读率
- 分享率和"在看"率
- 搜索来源关键词

**优化方向判断**：
- 推荐流量低 → 检查内容垂直度和互动率
- 搜索流量低 → 检查标题关键词和前200字
- 完读率低 → 检查文章长度和开篇质量
- 分享率低 → 检查是否有"社交货币"（金句/独家信息）

**内容-关键词映射表**：建立常写话题的关键词库，每篇文章覆盖对应关键词组合。如A股相关：牛市、大盘、沪指、创新高、成交额、主线方向等。

## 传播机制核心规律

### 道理 vs 新闻（转发与点赞的分水岭）

**转发的动机**是"我要让别人也看到这个"。点赞的动机是"这说的有道理"。

- **新闻属性**的内容让人转发——"你知道吗，公司一边裁人一边招了1万实习生"——这句话本身就是社交货币，转出去就是在传递信息
- **认知属性**的内容让人点赞——"原来公司是这么想的"——看完自己明白了，但没有转发给别人看的冲动。因为它不是"新闻"，是"道理"

**一句话总结：道理让人点赞，新闻让人转发。拉新靠转发。**

**选题公式：时事新闻钩子 + 认知升级内核**

- 裁员招实习生 = 新闻钩子（正在发生的事）+ 认知升级（为什么公司这么干）✅
- 纯加薪逻辑 = 只有认知升级，没有新闻钩子 ❌（会被认可但不会被传播）

**每篇文章动笔前必须回答两个问题**：
1. 这篇的"新闻钩子"是什么？
2. 读者转发的时候那句话是什么？

如果找不到，说明它可能会被认可但不会被传播——要么放弃，要么补一个新闻钩子。

### 转发钩子的六种类型（从传播心理学到实操）

转发的本质不是内容价值，而是社交价值。用户转发的不是文章，是这篇文章能帮他们在社交中购买的"形象"。

**转发层级 > 在看 > 点赞**，是社交认可度的阶梯。转发需要克服最高心理门槛，因此对内容的社交货币要求也最高。微信官方数据：61%的文章转发到朋友圈，80%的阅读量来自分享。

---

**类型一：替声型（Voice Proxy）——"你说出了我想说的话"**

文章替读者说出了他们想说但说不出（或不敢说）的话，转发即代言。

- **心理机制**：读者心里有一团情绪但提炼不成一句话，文章帮他提炼了。转发 = "这就是我的态度"
- **典型场景**：职场吐槽、社会不公评论、婚姻真相类文章
- **文章特征**：立场鲜明、语言犀利、金句密度高、有明确靶子但不点名
- **设计方法**：用极端化语言包装普遍情绪；设置反派角色让读者有共同对立面；口语化情绪化句式
- **案例**：咪蒙《致贱人》——替被道德绑架的读者发声，转发 = 立场表态
- **钩子位置**：金句独立成段，方便截图转发

**类型二：情绪唤起型（Emotion Activation）——"我读完忍不住要转"**

高唤醒情绪（敬畏/愤怒/兴奋/幽默）驱动转发，低唤醒情绪（满足/悲伤/平静）不驱动转发。

- **心理机制**：情绪到了峰值，不转发就等于把这股能量憋回去
- **数据**：激发敬畏感的内容转发率比基准高约30%；高唤醒情绪显著优于低唤醒情绪
- **典型场景**：震撼数据/故事、反转情节、集体行动类文章
- **文章特征**：前1/3处设情绪峰值；短句加粗感叹号强化情绪密度
- **案例**：新世相《逃离北上广》——兴奋+行动冲动，4小时刷屏
- **设计方法**：情绪波峰曲线：平静→冲突→高潮（转发点）→余韵

**类型三：社交货币型（Social Currency）——"转这个显得我有见识"**

转发该内容能让转发者在社交圈显得更有见识、更有趣或更有品位。

- **心理机制**：转发 = 购买社交形象。转出去后别人对转发者的评价 = 转发动机
- **子类型**：
  - 谈资型："你知道吗"——转发者成为信息源
  - 知识炫耀型："我懂得多"——科普辟谣类
  - 品位展示型："我有审美"——文化评论类
- **设计方法**：提供冷知识或反常识数据；用"90%的人不知道"包装信息；制作精美数据图表方便截图
- **案例**：支付宝年度账单——消费数据包装成个人生活档案，全民晒单

**类型四：代入型（Identity Substitution）——"想到一个具体的人"**

读者读完脑子里立刻浮现一个人或一个场景，转发给那个人或发到群里@他。

- **心理机制**：文章精准描述了一个读者认识的人/经历过的事。转发 = "你看这个人/这件事"
- **典型场景**：反驳型人格、控制欲父母、职场甩锅同事、伴侣沟通障碍
- **文章特征**：场景还原力强，每读一行读者都在点头；不写新闻写人性
- **设计方法**：对话场景还原（"你说…他说…"）；单句成段制造节奏感；结尾用"想到一个具体的人了吧"直接触发转发
- **案例**：心理类/人际关系类爆款——不需要新闻钩子，代入本身就是钩子
- **与新闻型的区别**：新闻型传播面更大（谁都想看），代入型传播更深（每个转发的人有强烈个人动机）

**类型五：利他型（Practical Value）——"这个对TA有用"**

纯粹的利他主义，认为内容对特定人或群体有价值。

- **心理机制**：转发 = 帮助。"这篇对我家人/同事/朋友有用"
- **典型场景**：育儿攻略、健康养生、行业报告、政策变化、避坑指南
- **文章特征**：明确受众指向；高实用性和可操作性；时效性强
- **设计方法**：文末用"转给需要的人""转给正在为XX发愁的人"
- **注意**：纯干货转发率不如社交货币型干货——需从"实用型"升级为"社交货币型"（思维框架化、观点锐化）

**类型六：身份认同型（Identity Signaling）——"转发这个我就是这种人"**

转发文章等于声明"我是哪类人"。

- **心理机制**：基于"可能自我理论"，用户通过转发构建和展示理想中的自己
- **典型场景**：深度思考文章（转发=有深度）、行业洞察（转发=专业）、群体认同（转发=我是同类）
- **文章特征**：明确的目标人群标签；"我们vs他们"对比框架；群体困境的精准表达
- **案例**：《北京有2000万人假装在生活》——每个在北漂的人看到后转发 = "我也是这个群体的一员"
- **不易转发的干货**：技能型（怕教会徒弟饿死师傅）、资源型（想独占）、工具型（想私藏）

---

### 转发钩子的四层布局

**第一层：标题钩子（一级钩子）**

标题本身既是点击钩子也是转发钩子。读者转发时，标题就是转发语的核心。

- 情绪词前置 / 数字锚点 / 反常识 / 身份标签
- 检验：读者转发时只看标题会不会点？

**第二层：开头情绪峰值钩子（前300字）**

在文章前1/3处必须设第一个情绪高点，防止读者完读前流失。

- 第一个震撼故事/数据放在前300字
- 短段落、加粗、节奏感制造视觉冲击
- 冲突-反转叙事结构

**第三层：文中金句钩子（截图钩子）**

最关键的转发钩子设计位置。

- 每800-1000字设置1-2个可截图转发的金句
- 金句独立成段，方便截图
- 加粗处理核心观点
- 在情绪最高点附近设置转发触发点

**第四层：结尾引导钩子**

自然引导，不生硬。

- 情绪煽动式："如果你也认同，请转发给更多人看到"
- 社交价值式："转给需要的人""这份清单值得团队每个人看"
- 身份绑定式："如果你是XX，请转发""所有XX都应该转一次"
- 代入触发式："想到一个具体的人了吧"
- 注意合规：不能强制分享或利益诱导

### 转发预判清单（动笔前必查）

1. 这篇文章能让转发者在朋友圈获得什么社交形象？
2. 读者转发时配文会写什么？（答不出 = 社交货币不够）
3. 这篇的转发钩子属于六种类型中的哪一种？
4. 转发后别人会怎么评价转发者？（正评价 = 会转，负评价 = 不转）

## 爆款公式（从实战验证的可复制模式）

### 大厂裁员爆款文章验证的完整公式

**转发公式**：具体的恐惧 × 权威的数据 × "不是你的错"的解脱感 × 让转发者显得有见识

**六要素（缺一不可）**：

**① 反直觉开头** → 让人停下来
- 用一个读者以为知道但实际不知道的数据切入
- 例："2025年Q1，大厂裁员XX万" → 读者以为知道，但你给出一个他没想到的角度
- 关键：不是"又要裁了"，而是"这次跟你想的不一样"

**② 揭秘结构** → "我来告诉你真相"
- 读者以为是A，你告诉他是B
- 例：裁员不是经营不善，是财务策略；高校砍专业不是教育新闻，是国家意志
- 关键：给读者一个"原来如此"的认知升级

**③ 数据表格** → 建立权威感
- 用对比表格让读者一眼看懂
- 例：被砍专业 vs 新增专业；旧岗位 vs 新岗位
- 关键：数据要具体、可查、有冲击力

**④ 核心爆破句** → 说出读者心里最想被说出的话
- 整篇文章最重要的一句话，让人想截图转发
- 标准：读完让人想哭/想截图/想发给朋友说"就是这个"
- 例："你没做错任何事，这是公司的财务策略"
- 例："你没落后，是人才生产线换了"
- **打在心上，不是打在脑子上**

**⑤ 愤怒出口** → 有靶子但情绪能落地
- 给读者一个愤怒的出口，但不能指向人
- 例："不是你选错了，是时代不需要了"（愤怒指向趋势）
- 关键：愤怒之后要有解脱感，否则读者会抵触

**⑥ 暖收尾** → 读完是正向的才会转发
- 收尾必须让读者有东西可以拿走
- 例："你比你以为的值钱，只是有人不愿意付这个价"
- 例："这38个专业就是未来10年的招聘JD预览版。你现在能看懂几个，决定了你三年后能不能看懂那批新人的简历"
- **禁止**："时间会给答案"——等于什么都没说

### 标题要有冲突感 = 制造认知缺口

弱标题：纯描述性（"大厂裁员分析"）
强标题：制造冲突（"大厂裁员进行时·大逃杀开始了"）

**标题三重检验**：
1. 读者看到标题会不会停下来？
2. 标题是否传递了一个明确的判断或事实？
3. 标题里有没有读者会搜索的关键词？

### 内容三要素（信息增量的来源）

**① 有趣的故事和案例**
- 每篇文章至少有一个具体的故事/案例/场景——不是抽象概念堆砌
- 案例必须是读者能代入的（"我朋友""我前同事""上周看到一个帖子"）
- 故事的作用：让读者放下防备，愿意继续读下去
- 好故事 > 好道理。道理人人会讲，故事才是护城河

**② 反常识的结论**
- 每篇文章至少有一个让读者"没想到"的判断
- 读者以为A，你说其实是B——这就是信息差，是让人转发的核心动力
- 反常识不是瞎编，是基于事实挖到更深一层的真相
- 例："高校砍专业不是教育新闻，是产业出清的确认信号"
- 例："计算机就业率82.4%，看起来不低？但全国均值是86.7%"

**③ 引发共鸣的内容**
- 说出读者心里有但嘴上说不出的话——这是爆破句的本质
- 共鸣 ≠ 煽情。共鸣是"你精准描述了我的处境"
- 判断标准：读者看完想@朋友说"你看，是不是就是咱们"
- 共鸣的对象必须精准（大厂人/程序员/30+职场人），不能泛泛而谈

**三要素的关系**：故事让人读下去 → 反常识让人"卧槽原来是这样" → 共鸣让人转发。缺故事则干巴，缺反常识则平庸，缺共鸣则不传播。

### 写作核心要求

**每篇文章必须有"爆破句"**
- 让人想截图转发的话
- 短、狠、不解释，让读者自己品
- 打在心上（情绪），不是打在脑子上（逻辑）

**结尾加转发钩子**
- 不只温暖收尾，要给一个具体的判断
- 让转发的人显得有见识、有深度
- 禁止"时间会给答案"式的空话

**每段都要有"你"**
- 全文从第一段就开始跟读者对话
- 不要到第六节才出现"你"
- 读者要觉得文章在说自己

## 写作避坑指南（从实战教训总结）

### 说教感规避（最高优先级，违反此条等于废稿）

**核心原则**：说教味是公众号文章的毒药。读者一旦感觉你在"教他做人"，立刻划走，而且再也不会回来。

**绝对红线**：

- 不能让读者觉得你在居高临下
- 不能替读者做判断（"你应该……""你至少……"）
- 不能替读者算账（"你少赚了……""你错过了……"）
- 不能质问读者（"你看过吗？""你想过没有？"）
- 不能用"教导"语气总结（"所以你要……""记住……"）
- **不能密集使用问句和反问句**——连续出现两个以上问句，读者感觉在被审问，说教味立刻上来
- **不能每段都以"你"开头**——"你"可以用，但全文"你"的密度要控制，不能形成"你怎样你怎样你怎样"的节奏。读者是在听故事，不是在被点名

**为什么说教味致命**：

- 读者来看文章是为了获取信息，不是来被教训的
- 说教暗含"我比你懂"的优越感，触发读者的防御心理
- 读者一旦反感，不会留到结尾，更不会转发——完读率和分享率双杀

**说教的写法（禁止）**：

- "你面试新人的时候，会发现……"
- "你带团队的时候，应该注意……"
- "你想必已经感受到了……"
- "你应该看到这个趋势……"
- "你去年如果只买了XX，没买XX——你至少少赚了……"
- "你平时看过它们吗？"
- "你以为你买的是XX，其实你买的是XX"

**平聊的写法（推荐）**：

- "有个事挺有意思……"
- "值得多说两句……"
- "回到最开始的问题……"
- "你看这个数据……"
- "大多数人发现它，是在它已经涨了一倍之后。"（观察，不指责）
- "这几个名字，大多数人是在它们涨完之后才第一次听说。"（陈述事实，让读者自己品）

**自检方法**：写完后读一遍，如果感觉像在"教读者做事"，就是太重了。如果感觉像在"跟朋友聊一个发现"，就对了。

**终极判断**：把每句话想象成你在饭桌上跟朋友说。如果朋友会翻白眼说"行了行了别教我了"，这句话就必须改。

### 不要总是挂同一件事

- 刚写了裁员，下一篇就不要挂裁员
- 读者会烦，会产生审美疲劳
- 同一个话题可以换角度：趋势、信号、风向、人才供应链切换
- 情绪是"发现新大陆"而不是"又要完了"

### 明确目标人群

- 一篇文章不可能覆盖所有人
- 写之前先确定：这篇文章到底写给谁看？
- 每个群体的爆破句完全不同
- 覆盖所有人 = 谁都没打中

### "你"必须在前两段出现

- 如果全文在说"高校"、"专业"、"政策"，到后面才出现"对职场人意味着什么"，太晚了
- 读者已经走了一半
- 第一段或第二段就要把"你"放进去
- 让读者觉得"这篇文章在说我"

### 核心爆破句的标准

**不够狠的爆破句（打在脑子上）**：
- "高校砍专业不是先导信号，是确认信号"——读完反应是"哦有道理"，然后划走

**够狠的爆破句（打在心 上）**：
- "你花四年学的那个专业，教育端已经决定停产了"——读完想截图
- "你没落后，是人才生产线换了"——读完想转发
- "不是你选错了，是时代不需要了"——读完想哭

**判断标准**：爆破句让读者的反应是"哦有道理"还是"就是这个"？前者不够狠。

### 收尾要有立场

**软的收尾（禁止）**：
- "至于这38个新专业到底有几斤几两——时间会给答案"
- 读者读完不知道该怎么办，也没有转发的理由

**有立场的收尾（推荐）**：
- "这38个专业，就是未来10年的招聘JD预览版。你现在能看懂几个，决定了你三年后能不能看懂那批新人的简历"
- 给读者一个具体的判断，让转发者显得有见识

### 最重要的素材要放在最显眼的位置

- 五部门联合发文是全文最好的角度，不能放在第四节
- 应该在开头或标题里就出现
- 制造"原来这么严重"的冲击感

## 每日热点速评类文章格式

### 与深度文章的区别
每日热点速评是轻量级日更内容，不用深度文章的复杂排版模块（dataCard、highlightBox等），采用极简格式。

### 格式规范
- **标题**：包含当日最大热点+可搜索关键词（如"A股突破4200创11年新高，这次是真牛市"）
- **正文**：4-6个编号段落（1. 2. 3. 4. 5.），每段一个热点
- **编号格式**：阿拉伯数字（1,2,3），不用中文数字
- **无章节标题**：不要给每个热点加小标题，直接inline编号
- **无分隔线**：段落之间不要divider，靠段间距分隔
- **无CTA框**：不用ctaBox，文末可以有简单评论引导
- **结尾**：一句评论引导问题（如"你觉得A股这波能到4500吗？评论区说说。"）
- **不要"全文完"**

### 内容编排原则
- **主打话题放第1位**：平台给文章打标签主要看第1个话题和前200字
- **相关话题相邻**：有关联的话题放一起
- **每段信息密度高**：给事实、给数据、给判断，不铺垫不废话
- **每段有观点**：不做纯新闻搬运，要有"老炮"式的直接判断

### 话题选择原则（猫笔刀流量逻辑）
- **信息差**：读者不知道但想知道的事
- **半句话**：说一半留一半，让人想评论补充
- **金句**：提供可转发的观点（社交货币）
- **身份认同**：让读者觉得"这是我的想法"

## 写作流程

1. **确认选题**：和用户讨论文章主题、核心观点
2. **确认脱敏**：询问是否需要脱敏处理，脱敏级别
3. **确认标题**：提供3-5个标题选项让用户选择
4. **理结构**：先输出文章大纲让用户确认
5. **写正文**：生成DOCX脚本并执行
6. **确认输出**：让用户查看DOCX，收集修改意见
7. **迭代修改**：根据反馈调整

## 内容分流策略（不同平台写不同内容）

**微信公众号（冲量、涨粉）**：
- 只发认知觉醒类、利益链拆解类、焦虑唤醒类
- 社交货币必须为正（转发 = 我有见识），绝不能为负（转发 = 我需要帮助）
- 不教方法，只揭真相。让读者自己悟，不替读者做总结
- 方法论类、实操类文章不发公众号——说教味天然重，社交货币天然负

**内网技术社区（口碑、影响力）**：
- 方法论类、实操类、经验沉淀类都可以发
- 说教可以接受，读者主动来学东西
- 可以写得细、写得长、可以给四步行动方案

**一鱼多吃的正确姿势**：
- 同一个选题，两种写法，不是同一篇文章发两个地方
- 公众号版：只拆逻辑，不给方案（"你工作5年没升职，不是能力问题，是你把自己当商品但从来没上架"）
- 内网版：给框架、给步骤、给模板（"四步把隐形思考可见化"）
- 判断标准：这篇文章发到朋友圈，别人会觉得"这人有见识"还是"这人需要帮助"？后者不发公众号

## 冷启动期选题纪律

**不打热点正面**：
- 40粉丝的号跟百万大号在同一热点上抢推荐流量池，算法不会选你
- 热点可以蹭，但必须用侧面角度切入——不写"英伟达财报"，写"你的AI持仓为什么没涨"
- 发之前搜微信"搜一搜"，看同话题有多少大号在写。头部号铺满了，换角度或换选题

**情绪优先于干货**：
- 干货文让人"涨知识了"→收藏→不转发→没有二次传播
- 情绪文让人"这说的就是我"→转发→社交裂变
- 冷启动期增长全靠转发，所以情绪优先
- 收藏不产生传播，转发才产生传播

**文章长度上限**：
- 冷启动期（粉丝<1000）单篇不超过1500字
- 超过2000字完读率大概率低于50%，被算法卡在冷启动池出不去
- 干货多就拆系列，每篇控制长度

**受众不能飘**：
- 核心受众是大厂/职场人（20-40岁）
- 每篇发之前问自己：这群人看完会觉得"这说的就是我"吗？
- 如果答案是"涨知识了"而不是"就是这个"，选题方向要调

## 内容生产纪律（学刘润的产品化思维）

### 日更节奏与内容配比

**周产出标准**：
- 深度长文：每周2-3篇（周二、周四、周六发布）
- 热点速评：每天1篇（早上7:00-9:00发布）
- 金句卡片：每篇文章提取1-3条，当天发朋友圈/社群

**内容类型配比**：
- 60% 热点速评（日更保活跃度+标签强化）
- 30% 深度拆解（核心内容，建立专业壁垒）
- 10% 互动/观点碰撞（制造讨论，提升互动率）

### 写作质量自检表

**每篇文章发布前过这张表**：

| 维度 | 检查项 | 标准 |
|------|--------|------|
| 逻辑 | SCQA完整性 | S/C/Q/A四要素是否都有？ |
| 深度 | 利益链拆解 | 是否拆出了"谁赚钱/谁买单"？ |
| 情绪 | 情绪浓度 | 开篇和结尾是否达到Level 4？ |
| 可读 | 娱乐性 | 每500字是否有1个类比/段子/荒诞对比？ |
| 数据 | 具体性 | 是否有≥2组具体数据（金额/百分比/倍数）？ |
| 传播 | 金句提取 | 能否提炼出1句可独立传播的金句？ |
| SEO | 关键词 | 标题和前200字是否包含可搜索关键词？ |
| 合规 | 红线检查 | 是否触碰六大违规类型？数据是否可查？ |
| 引流 | CTA | 结尾是否有损失感驱动的关注引导？ |
| 复用 | 一鱼多吃 | 是否已提取金句/讨论话题/明日切入点？ |

### 选题储备系统

**选题四象限**：
- **高热度+高匹配**（必写）：当日热点与投资思维/人生决策直接相关
- **低热度+高匹配**（蓄力写）：常青话题，不赶时效，慢慢打磨成深度长文
- **高热度+低匹配**（蹭角度）：热点本身与定位无关，但可以用"投资思维"切入
- **低热度+低匹配**（不写）：与定位无关且无流量

**选题来源**：
- 实时热搜（用`hot-topics`skill获取）→ 判断属于哪个象限
- 财经数据异动（GDP、CPI、股市、楼市）→ 天然高匹配
- 社会事件（教育、就业、消费降级）→ 用投资思维切入
- 读者评论区高频问题 → 痛点验证过的选题
