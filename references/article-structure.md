# 文章结构模板

## 标题
- **主标题**：狠、短、有冲击力，最好是一个判断句或反问句
- **副标题**：补充说明文章主题，带悬念感
- 示例：《你才是那个赌徒》《你的钱是怎么没的》《你不醒，钱就是别人的》

## 正文结构（不超过5-6个章节）

**底层逻辑**：SCQA确保逻辑完整，利益链拆解确保内容有料，情绪浓度确保读者读完。

1. **开篇钩子（S+C）**：用震撼数字或荒诞场景建立情境，立刻引入冲突。情绪浓度≥Level 4。
   - 示例："2025年，AI替代了43万个白领岗位（S）。但最先失业的不是能力最差的，而是工资最高的那批（C）。"
2. **重新定义（Q）**：打破读者固有认知，提出读者从没想过但一听就睡不着的问题。
3. **撕破伪装（拆解链条）**：把利益链条一环一环拆开，每个环节给数据、给对比、给类比。情绪浓度≥Level 3。
4. **算账环节（数据暴击）**：用具体数据算几笔让人睡不着觉的账。精确到具体金额/百分比/倍数。
5. **核心观点（A）**：扣哥的判断和方法论，用一句可独立传播的金句封装。情绪浓度≥Level 4。
6. **CTA引流**：制造损失感+关注引导。情绪浓度≥Level 4。

**自检清单**：
- [ ] S有没有？读者能不能在3秒内代入情境？
- [ ] C够不够痛？冲突是否直接戳到读者的处境？
- [ ] Q够不够尖锐？问题是否让读者"从没想过但一听就慌"？
- [ ] A够不够实？答案是否能提炼成一句可传播的金句？
- [ ] 利益链条拆了没有？读者看完是否知道"钱去了哪"？
- [ ] 情绪浓度够了没？开篇和结尾是否达到Level 4？
- [ ] 有没有至少1个"笑完冒冷汗"的类比或段子？

## 结尾CTA模板

```
你亏钱，是因为你从来不算账。

我每天一个投资认知，帮你比别人早半步看到机会。

关注我。
明天那条，别错过。
```

或更短版本：

```
每天一个认知，关注我，别再错过。
```
