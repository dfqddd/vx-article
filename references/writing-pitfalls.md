# 写作避坑指南（从实战教训总结）

## 说教感规避（最高优先级，违反此条等于废稿）

**绝对红线**：
- 不能让读者觉得你在居高临下
- 不能替读者做判断（"你应该……""你至少……"）
- 不能质问读者（"你看过吗？""你想过没有？"）
- 不能用"教导"语气总结（"所以你要……""记住……"）
- **不能密集使用问句**——连续两个以上问句=审问，说教味立刻上来
- **不能每段都以"你"开头**——全文"你"的密度要控制

**说教的写法（禁止）**：
- "你面试新人的时候，会发现……"
- "你去年如果只买了XX——你至少少赚了……"
- "你以为你买的是XX，其实你买的是XX"

**平聊的写法（推荐）**：
- "有个事挺有意思……"
- "你看这个数据……"
- "大多数人发现它，是在它已经涨了一倍之后。"

**终极判断**：把每句话想象成你在饭桌上跟朋友说。如果朋友会翻白眼说"行了行了别教我了"，这句话就必须改。

## AI味排除（写完必查，有一条就改）

1. **问句当小标题** → 删掉问句，直接陈述
2. **大量markdown格式感** → 用连贯自然段落叙述
3. **大量断言无论证** → 每个判断后面跟具体事实或数据
4. **没有主线** → 写之前定一条主线，每段都在推进这条线
5. **AI高频表达** → 删掉"值得一提""毋庸置疑""综上所述"等
6. **先问后答/自问自答** → 直接陈述结论，不要先提问再解答
7. **平行对仗/公式压缩** → 拆成自然叙述，只在爆破句位置偶尔用一次

**自检**：全文问号≤3个。问完立刻自己回答=AI味，删问号改陈述。

## 其他避坑

- **不总是挂同一件事**：同一话题换角度（趋势、信号、风向）
- **明确目标人群**：覆盖所有人 = 谁都没打中
- **"你"必须在前两段出现**：让读者觉得"这篇文章在说我"
- **核心爆破句打在心上**：反应是"就是这个"而非"哦有道理"
- **收尾要有立场**：给读者一个具体判断，禁止"时间会给答案"
- **最重要的素材放最显眼位置**：五部门联合发文不能放在第四节
