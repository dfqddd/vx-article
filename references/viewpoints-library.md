# 核心观点库（可复用）

## 投资思维类
- 你不是不投资，你是闭着眼睛把人生最大的钱投出去还不算账
- 真正的赌徒不是进股市的，是连算都不算就把人生押出去的人
- 风口从来不等人，它只奖励提前看到趋势的人
- 你不学投资思维，就算侥幸踩中一波风口，下一波你还是会错过
- 你的钱是怎么没的？因为你从来没算过账
- 工作占了你清醒时间的80%以上，这是你人生最大的一笔投资
- 信息差是猎人最重要的装备
- 晚一年的代价不是少赚一点，是后面几十年都在追

## 流量与认知类
- 互联网是有记忆的，捂嘴时代已经过去了
- 流量时代，没有一所学校能独善其身
- 林子大了什么鸟都有，但害群之马毁的是整个林子
- 不良媒体是流量的奴隶，已经偏离了事实本身
- 口碑反转需要很长时间，但毁掉只需要一天
- 每个学校都有那么几个败类，不可避免，但影响是真实的
