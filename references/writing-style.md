# 行文风格

## 核心原则
- **制造焦虑和紧迫感**：让读者意识到自己一直在"投资"（工作、婚姻、买房、生娃、选行业），但从来不计算ROI
- **戳破自我安慰**：直接撕开"我没亏只是没赚""等等看再说"的心理防线
- **老炮口吻**：不讨好读者，不写正确的废话，每句话都要有刺痛感
- **用数据说话**：具体数字、具体倍数、具体场景，不写空泛的道理

## 写作禁忌
- 不写"建议""可以""或许"这类软绵绵的词
- 不写正确的废话（如"投资有风险""要理性决策"）
- 不做老好人式的平衡论述
- 不用"首先、其次、最后"这种学术式结构

## 语言特征
- 短句为主，节奏感强
- **少用问句和反问句**——问句多了像在审问读者，说教味很重。陈述事实比质问更有力量
- **少用"你"密集出现**——"你"可以用，但不能每段都以"你"开头，不能连续出现
- 善用对比制造荒诞感（如买房被夸稳重 vs 炒股被骂赌博）
- 关键结论用**加粗+红色**标出
- 以观察者和讲述者的姿态写作，让事实和逻辑自己说话

## 三大技法融合（半佛仙人×刘润×余哥牛弹琴）

扣哥的风格 = 半佛的拆解刀法 + 刘润的内容框架 + 余哥的情绪浓度，三者融合而非照搬。

### 技法一：利益链条拆解法（学半佛仙人）

**核心**：任何现象背后都有一条利益链条，写文章就是把这条链条一环一环拆给读者看。

**拆解四步法**：
1. **定义利益方**：这件事里谁赚钱？谁买单？谁被割？
2. **追溯资金流**：钱从谁的口袋流向谁的口袋？中间经过几道手？
3. **识别信息差**：利益方之间的信息不对称在哪？谁故意制造信息差？
4. **暴露荒诞性**：把利益链条摊开后，读者会发现自己一直在为别人的利润买单

**扣哥适配**：半佛拆商业骗局，扣哥拆人生决策的隐性成本。

**句式示例**：
- "你以为你在选行业？不，你在为别人的退出通道买单。"
- "这笔钱经过了三道手才到你面前，每道手抽走20%，你算过没有？"
- "表面上是教育投资，拆开看是恐惧变现。"

### 技法二：深度+娱乐双满足（学半佛仙人）

- **类比降维**：用读者熟悉的场景解释复杂概念
- **黑色幽默**：在严肃分析中插入荒诞类比，制造"笑完冒冷汗"的效果
- **段子化表达**：每500字至少一个让人会心一笑或倒吸冷气的表达
- **"先爽后痛"节奏**：先用有趣的切入让读者读下去，然后在中段用数据拍醒

### 技法三：SCQA内容框架（学刘润）

- **S（Situation/情境）**：读者身处的真实场景
- **C（Complication/冲突）**：打破这个场景的变量
- **Q（Question/问题）**：冲突引出的核心问题
- **A（Answer/答案）**：扣哥的核心观点和方法论

SCQA是骨架，利益链拆解是血肉，情绪浓度是皮肤。不是死板模板，用它检查文章逻辑是否完整。

### 技法四：情绪浓度拉满（学余哥牛弹琴）

**情绪浓度四级**：
- **Level 1 陈述**："房价涨了30%。"（纯信息）
- **Level 2 带刺**："房价涨了30%，你的工资涨了吗？"
- **Level 3 戳心**："房价涨了30%，你的工资没涨，你还觉得自己没亏？"
- **Level 4 暴击**："房价涨了30%，你的工资原地踏步，你每天加班12小时的结果是——帮房东多赚了一套车位。"

**扣哥标准**：正文≥Level 3，核心段落必须Level 4。开篇和结尾必须Level 4。

**禁区**：不人身攻击、不用民族主义煽动、不编造数据。

### 技法五：一鱼多吃内容复用（学刘润）

- **深度长文**（1200-2000字）→ 完整论述
- **热点速评**（500-800字）→ 核心观点+当日热点
- **金句卡片** → 文中最戳心的一句话
- **评论区互动话题** → 从争议点提取讨论问题
