# 每日热点速评格式

## 与深度文章的区别
轻量级日更内容，不用复杂排版模块，采用极简格式。

## 格式规范
- **标题**：包含当日最大热点+可搜索关键词
- **正文**：4-6个编号段落（1. 2. 3.），每段一个热点
- **编号格式**：阿拉伯数字，不用中文数字
- **无章节标题**：直接inline编号
- **无分隔线**：靠段间距分隔
- **无CTA框**：文末简单评论引导
- **结尾**：一句评论引导问题
- **不要"全文完"**

## 内容编排原则
- 主打话题放第1位（平台打标签看第1个话题和前200字）
- 相关话题相邻
- 每段信息密度高：给事实、给数据、给判断
- 每段有观点：老炮式直接判断

## 话题选择原则
- **信息差**：读者不知道但想知道的事
- **半句话**：说一半留一半，让人想评论
- **金句**：提供可转发的观点
- **身份认同**：让读者觉得"这是我的想法"
