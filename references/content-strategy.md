# 内容策略体系

## 内容分流策略

**微信公众号（冲量、涨粉）**：
- 只发认知觉醒类、利益链拆解类、焦虑唤醒类
- 社交货币必须为正（转发=我有见识）
- 不教方法，只揭真相
- 方法论类/实操类不发公众号（说教味天然重）

**内网技术社区（口碑、影响力）**：
- 方法论类、实操类、经验沉淀类都可以发
- 可以写得细、写得长

**一鱼多吃**：同一选题两种写法，不是同一篇文章发两个地方。

## 冷启动期选题纪律

- **不打热点正面**：40粉丝跟百万大号抢流量池，算法不会选你。用侧面角度切入。
- **情绪优先于干货**：干货→收藏→不转发；情绪→转发→社交裂变
- **文章长度上限**：粉丝<1000时单篇≤1500字
- **受众不能飘**：核心受众大厂/职场人(20-40岁)

## 内容生产纪律

**周产出**：深度长文2-3篇 + 热点速评每天1篇 + 金句卡片每篇1-3条

**配比**：60%热点速评 + 30%深度拆解 + 10%互动/观点碰撞

**质量自检表**：
| 维度 | 标准 |
|------|------|
| SCQA | S/C/Q/A四要素都有？ |
| 利益链 | 拆出"谁赚钱/谁买单"？ |
| 情绪 | 开篇和结尾≥Level 4？ |
| 可读 | 每500字有1个类比/段子？ |
| 数据 | ≥2组具体数据？ |
| 传播 | 能提炼1句可独立传播的金句？ |
| SEO | 标题和前200字含可搜索关键词？ |
| 合规 | 未触碰六大违规？数据可查？ |

## 选题四象限

- **高热度+高匹配**（必写）
- **低热度+高匹配**（蓄力写，常青话题）
- **高热度+低匹配**（蹭角度）
- **低热度+低匹配**（不写）

## 选题二维模型（爆发力 × 长尾力）

- 爆发力由完读率决定，长尾力由"内容会不会过期"+"受众盘子大小"决定
- **右上角（高爆发+有长尾）= 超级文**：职场账本类最该死磕（常青爆款会复利）
- **左上角（高爆发+无长尾）= 时效弹药**：阶段性破圈拉新
- **右下角（低爆发+有长尾）= 慢滚雪球**：慢慢滚但一直有量
- **左下角 = 两头不占**：最不划算

每个选题过两道关：第一关"它会过期吗"，第二关"受众盘子多大"。两关都过的优先写。
