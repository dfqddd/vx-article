# 合规红线与脱敏处理

## 六大违规类型（绝对不碰）
1. **政治敏感**：不涉及政治人物、政治事件、政府政策评价
2. **色情低俗**：不写招嫖、情色、性暗示内容
3. **虚假谣言**：所有数据和事件必须有据可查，不编造、不夸大
4. **商业欺诈**：不推荐股票、不喊单、不传销、不网赚
5. **恶意诱导**：不用"不转不是中国人"等胁迫分享
6. **侵权冒用**：不冒用他人身份，不盗用他人内容

## 标题党红线
- 标题可以狠，但不能虚假夸大或与内容严重不符
- 例：《你才是那个赌徒》✅；《震惊！某某死了》❌

## 处罚机制（阶梯式）
初犯删文3天 → 较重封号7天 → 90天3次封号15-30天 → 永久封号

## 内容写作底线
- 只讲事实，不编造、不歪曲
- 可以评论，但不人身攻击
- 涉及具体人物时用"某"字脱敏
- 投资类内容必须加免责声明

## 脱敏处理

**行文前询问用户是否需要脱敏。**

- **轻度**：人名用"某"（余某、杨某），企业保留全称
- **中度**：学校用"某高校"，企业用"某品牌"
- **重度**：所有具体名称都用代称

用户说保留的例外保留。
